Getting started

An overview of Bauhaus, how to install and use, basic templates and examples, and more.

Installation

To install Bauhaus in your latest and greatest Laravel project, use composer and simply add krafthaus/bauhaus to your composer.json file:

"require": {
	"laravel/framework": "4.1.*",
	"krafthaus/bauhaus": "dev-master"
}

Then use composer to install the package.

$ composer update

Registering

Once installed, you have to register the service provider in app/config/app.php the providers array:

'providers' => array(
	'KraftHaus\Bauhaus\BauhausServiceProvider'
)

Configuration

Once everything is installed and registered, you need to tell composer where to find the bauhaus files. Add the admin folder to the app/ directory and put the following line in your composer.json file: 

"autoload": {
	"classmap": [
		"app/admin"
	]
}

Then publish the config file with php artisan config:publish krafthaus/bauhaus. This will add the main bauhaus config file in your application config directory.

And last but not least you need to publish to package's assets with the php artisan asset:publish krafthaus/bauhaus command.
