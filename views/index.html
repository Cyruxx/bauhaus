Lists

The create a list view, there are several (but easy) steps to take. First, if you did not create your Bauhaus controller yet, there is a quick introduction located here.

When you open your fresh, newly created Bauhaus file, you'll see a couple of pre-defined methods. For now, we'll concentrate on the configureList() method.

By default, the configureList method is passed a $mapper variable to it. This variable contains an KraftHaus\Bauhaus\Mapper\ListMapper instance that handles everything from adding new fields to passing around other instances needed to render the list view.
